
# Weather Forecast Web Application

A responsive Weather Forecast Web Application built with HTML, CSS, and JavaScript using the OpenWeather API. It provides real-time weather updates, including temperature, humidity, wind speed, and weather conditions, with dynamic icons, input validation, and error handling.


## Features

- Real-Time Weather Search – Search and view current weather information for any city worldwide using the OpenWeather API.
- Live Weather Details – Displays real-time temperature, weather condition, humidity, and wind speed.
- Dynamic Weather Icons – Automatically updates weather icons based on conditions such as Clear, Clouds, Rain, Mist, and Snow.
- Input Validation & Error Handling – Prevents empty searches and displays a "Location Not Found" message for invalid city names.
- Responsive & Interactive Interface – Provides a clean, user-friendly design with support for both the Search button and Enter key for quick weather searches.


## Tech Stack

**Frontend:** HTML5, CSS3, JavaScript (ES6)

**Tools:** Visual Studio Code
